解决Android Studio 3.3 一直卡在“Download https://services.gradle.org/distributions/gradle-4.10.1-all.zip”

三月 31, 2019 发布在 Android, Android Studio

解决Android Studio 3.3 同步项目时，一直卡在下载Gradle中

问题描述

重新安装Android Studio 3.3，项目同步时，一直卡在“Download https://services.gradle.org/distributions/gradle-4.10.1-all.zip”

问题原因

网络问题，应该是gradle服务器被墙

解决方法

  1. 先退出Android Studio，在浏览器手动下载gradle， https://services.gradle.org/distributions/gradle-4.10.1-all.zip

  2. 下载gradle包后，校验sha256，地址https://services.gradle.org/distributions/gradle-4.10.1-all.zip.sha256

    certutil -hashfile D:\DOWNLOAD\gradle-4.10.1-all.zip

  3. 将文件gradle-4.10.1-all.zip复制到C:\Users\<username>\.gradle\wrapper\dists\gradle-4.10.1-all\bzyivzo6n839fup2jbap0tjew文件夹下，无需解压，其中bzyivzo6n839fup2jbap0tjew文件每个人的可能不一样

  4. 重新打开Android Studio 同步项目
