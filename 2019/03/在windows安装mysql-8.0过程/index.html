在Windows安装MySQL 8.0过程

三月 6, 2019 发布在 数据库, MySQL

在Windows安装MySQL 8.0 的步骤，使用的是免安装版

下载

  1. 确保Windows登录的是管理员账号
  2. 前往官方下载地址，翻到页面最下方，选择Windows (x86, 64-bit), ZIP Archive，点击Download下载
  3. 解压ZIP文件到某个目录下，如"D:\Program Files”

创建配置文件

在启动MySQL服务器时，可以指定参数，也可以在配置文件中进行配置。MySQL服务器启动时会从多个位置加载配置文件，例如C：\和MySQL安装目录

MySQL首先在my.ini文件中查找每个位置的选项，然后在my.cnf文件中查找。但是，为避免混淆，最好只使用一个文件。如果您的PC使用的引导加载程序中C：不是引导驱动器，则唯一的选择是使用my.ini文件。无论您使用哪个选项文件，它都必须是纯文本文件
