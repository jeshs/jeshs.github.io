Genymotion报错INSTALL_FAILED_CPU_ABI_INCOMPATIBLE的解决方法

二月 24, 2019 发布在 问题解决, Genymotion

解决在Genymotion模拟器上安装APP失败的问题

问题描述

在Genymotion模拟器上，安装部分APP失败，提示"INSTALL_FAILED_CPU_ABI_INCOMPATIBLE"

问题原因

Genymotion是使用的x86架构 所以如果安装的APP只支持ARM架构的手机，就无法正常安装

解决方法

可以在Genymotion 模拟器上安装ARM支持库

  1. 下载对应模拟器版本的ARM支持库，目前支持到Android 8.0
    • Android 5.0、6.0、8.0的下载地址，备用地址（资源来自网络，没有找到官方来源和作者，请自行判断是否可信）
    • Android 4.4及以下（资源来自XDA作者）
  2. 将对应版本的ARM支持库拖动到Genymotion模拟器中，点击OK。注意ARM支持库的所在路径不能包含中文或者空格
  3. 重启模拟器
