VirtualBox虚拟机不能ping通宿主机的解决方法

二月 24, 2019 发布在 问题解决, VirtualBox

解决VirtualBox虚拟机不能ping通宿主机的问题

问题描述

在VirtualBox虚拟中，无法ping通宿主机，宿主机可以ping通虚拟机

虚拟机的网络已经设置为“桥接网卡”，也可以正常上网

问题原因

Windows主机防火墙没有启用ICMPv4-in这个规则，被防火墙阻止

解决方法

  1. 打开Windows控制面板 -> 系统和安全 -> 高级设置
  2. 点击防火墙“入站规则”，找到“文件和打印机共享(回显请求 - ICMPv4-In)”，有两个配置，一个是“专用，公用”，另一个是“域”。右键启用“专用，公用”的那条规则即可
